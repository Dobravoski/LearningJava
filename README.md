# ☕ Java Studies

This repository is where I keep track of my progress learning Java, starting from the very basics and gradually moving into more practical and structured concepts.

Instead of putting everything into a single project, I split things into smaller ones based on what I’m studying. It makes things easier to manage and also helps me revisit specific topics without getting lost in a huge codebase.

This is not meant to be perfect or finished. It’s a space to learn, experiment, make mistakes, and improve over time.

---

## 📂 Projects

### 🔹 java-basics

This is where most of my foundation in Java was built.

Over time, it grew from simple programs into a broader set of exercises and experiments, including:

* basic syntax and first programs
* conditionals and loops
* object-oriented programming
* collections and data structures
* file and data handling
* streams and more functional-style code

It works more like a sandbox than a single project, where I test ideas and reinforce concepts through practice.

---

### 🔹 jdbc-mysql

This project is where things start to feel closer to real-world development.

Instead of focusing only on the language, the idea here is to understand how Java interacts with external systems, especially databases. Along the way, I explore things like:

* how JDBC works in practice
* managing database connections
* organizing code related to data access
* handling errors in a cleaner way
* keeping configuration outside the code

More than just connecting to a database, this project is about learning how to structure code in a way that is easier to maintain and scale.
