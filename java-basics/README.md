# Java Basics ☕

This folder is basically my Java playground.

It contains everything from my very first `System.out.println("Hello World")` to more advanced core topics like streams, file handling, and some early software design concepts.

## 💡 What's this about?

I'm using this space to learn Java by doing — writing small pieces of code, testing ideas, breaking things, and figuring out how they work.

It's not a single project, but a collection of exercises and experiments.

## 🧠 What I've been learning

- Java syntax and first programs
- Variables and data types
- Conditionals (`if`, `switch`)
- Loops (`for`, `while`, `do-while`)
- Arrays and matrices
- Object-Oriented Programming (OOP)
    - Classes and objects
    - Encapsulation
    - Inheritance & polymorphism
- Collections (`List`, `Set`, `Map`)
- Exception handling
- File handling (IO / NIO)
- Streams and lambda expressions
- Some basic software design ideas

## 🧪 How I study

- Writing small, focused exercises
- Testing one concept at a time
- Revisiting topics to understand them better

## 📂 Structure

The code is organized more or less by topic as I progressed through my studies.  
Nothing too fancy — just what made sense while learning.

## 🎯 Why this exists

- Build a solid Java foundation
- Keep track of my progress
- Have something to look back at when I forget how things work (which happens 😅)

## ⚠️ Note

You might find:
