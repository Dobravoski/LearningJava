# JDBC + MySQL 🛢️

This is where I’m learning how Java actually talks to a database.

Right now it’s just getting started, but the idea is to build a solid understanding of JDBC and how to work with MySQL in real applications.

## 💡 What's this about?

This project is focused on connecting Java to a database and handling real data.

It’s less about syntax and more about:
- how applications store data
- how to retrieve and manipulate it
- how to structure code around a database

## 🧠 What I'm working on (and will be working on)

- Setting up JDBC connections
- Understanding how `Connection`, `Statement`, and `ResultSet` work
- Basic CRUD operations:
    - Create
    - Read
    - Update
    - Delete
- SQL integration inside Java
- Handling exceptions and errors properly
- Organizing database-related code

## 🔜 Planned topics

- Prepared Statements
- Transactions
- DAO pattern
- Connection management improvements
- Better project structure
- Possibly moving towards frameworks later (like Spring)

## 🧪 Approach

- Start simple (just making things work)
- Then refactor and improve
- Try to understand what's happening under the hood
